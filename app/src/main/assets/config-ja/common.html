共通-設定

共通の設定

ファイル選択画面とイメージ表示画面で共通の設定です。

文字コード

ZIP又はRARファイル内のファイル名をどの文字コードとして扱うかを設定します。
Windowsで作成したZIPファイルはShift_JIS、Mac OS Xで作成したZIPファイルはUTF-8になっているようです。
  • UTF-8
  • Shift_JIS
  • EUC-JP

.で始まるファイルを表示しない

一部のOSで隠しファイルとして扱う”.”で始まるファイルを非表示にする設定です。
Mac OS Xで作成したZIPファイルの中に画像を格納した場合に作成される、”.”で始まり拡張子がjpg又はpngで画像ではないファイルを非表示にできる様にしました。

画面を回転するボタン

ファイル選択画面とイメージ表示画面で画面の向きを回転するためのボタンを設定します。
端末によりフォーカスボタンの有無やシャッターボタンが押しにくいなどがありますので、操作しやすいボタンを設定してください。
