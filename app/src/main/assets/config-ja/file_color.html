ファイル選択画面-色設定

ファイル選択画面の色設定

ファイル選択画面で使用する色を設定します。

プリセット色

プリセットの選択

アプリで用意している配色のプリセットを選択します。
好きな色を指定したい場合は「カスタム」を選択してください。

プリセットの反映

現在選択中のプリセットの配色をカスタム設定の各項目に反映します。

カスタム設定

プリセットの選択で「カスタム」を選択したときに有効になります。
画面を構成する個々の部品について、以下の設定項目で任意の色を設定できます。

サーバ選択のテキスト色

サーバ選択画面のサーバ名のテキスト色です。

ディレクトリの色

ファイル選択画面のディレクトリのテキスト色です。

未読ファイルの色

ファイル選択画面の未読の圧縮ファイルのテキスト色です。

読みかけファイルの色

ファイル選択画面の読みかけの圧縮ファイルのテキスト色です。

既読ファイルの色

ファイル選択画面の既読の圧縮ファイルのテキスト色です。

イメージファイルの色

ファイル選択画面のイメージファイルのテキスト色です。

ファイル情報の色

ファイル選択画面のファイル更新日付やファイルサイズのテキスト色です。
サーバ選択画面のURIのテキスト色です。

リストの背景色

ファイル選択画面のリストの背景色です。

タイトルのテキスト色

ファイル選択画面上部のファイルパスを描画しているエリアのテキスト色です。

タイトルのテキスト背景色

ファイル選択画面上部のファイルパスを描画しているエリアの背景色です。

ツールバーのボタン色

ファイル選択画面のツールバーのボタンの色です。
