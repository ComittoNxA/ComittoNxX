ファイル選択画面-設定

ファイル選択画面設定

ファイル選択画面の表示と操作に関連する設定です。

表示方向

ファイル選択画面を表示する向きを設定します。
カメラボタンによる回転は縦固定又は横固定の場合のみ動作します。
  • 回転あり
    • 本体の向きに合わせて自動的に画面を回転します。
  • 縦固定
    • 本体の向きに関係なく画面を縦向き表示に固定します。
  • 横固定
    • 本体の向きに関係なく画面を横向き表示に固定します。

ソート表示

ファイルリストの表示順番を設定します。
  • ソートなし
    • ファイル一覧を取得したそのままの順番で表示します。
  • ディレクトリ／ファイル混在
    • ディレクトリとファイル名を区別せず名前順で表示します。
  • ディレクトリ→ファイル順
    • ディレクトリとファイル名を分け、ディレクトリを先に名前順で示します。

Backキーの動作

ファイル選択画面でBackキーを押したときの動作を設定します。
  • ComittoNの終了
    • Backキーの二度押しでアプリを終了します。
  • 親ディレクトリに移動
    • ファイルリストの親ディレクトリに移動します。
    リスト中の「..」やツールバーの「上へ」を選択と同じです。
  • 前のディレクトリに戻る
    • ディレクトリを移動した履歴を逆に戻っていきます。

フォントサイズ

ファイル名を描画するフォントサイズを設定します。

サマリのフォントサイズ

ファイル更新日付とファイルサイズを描画するフォントサイズを設定します。

ツールバー表示

ファイル選択画面にツールバーを表示します。

サムネイル表示

画像ファイルを縮小しサムネイルとしてリストに表示します。
圧縮ファイルの場合は先頭の画像を表示します。

サムネイルのサイズ

サムネイルの描画サイズを設定します。
  • 最小（64x48）
  • 小（120x90）
  • 中（160x120）
  • 大（200x150）
  • 最大（240x180）
