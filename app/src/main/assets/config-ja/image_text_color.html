イメージ＆テキスト表示画面-色設定

イメージ＆テキスト表示画面の色設定

イメージ表示とテキスト表示画面で使用する色を設定します。

イメージ表示とテキスト表示共通の設定

周囲の余白の色

画像又はテキスト周囲の余白の色を設定します。

中央の余白の色

「並べて表示」のときの２ページの画像又はテキストの間の色を設定します。

情報表示の背景色

メニュー選択やページ選択を選択中の表示に使用される背景色を設定します。

テキスト表示の設定

テキストの色

テキスト表示の文字の色を設定します。

背景の色

テキスト表示の背景（用紙）の色を設定します。

背景の色（グラデーション用）

背景（用紙）のグラデーション設定を有効にしたときに使用するの色を設定します。
上記「背景の色」との２色でグラデーション表示を行います。

グラデーション設定

背景のグラデーション表示を設定します。
グラデーションなし又はグラデーションの方向をリストから選択します。
