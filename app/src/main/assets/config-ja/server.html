サーバ選択画面-設定

サーバ選択画面の説明

サーバの登録

リスト中の設定したい位置の項目を長押しすると設定画面が表示されますので、サーバの情報を入力してください。
設定済みのリスト項目であれば設定されている内容が表示されます。
  • 表示名
    • リストへの表示にのみ使用する名称です。
    この項目を設定していない場合はリストに「未定義」と表示されますので、人に見せたくないサーバであれば使用するときだけ表示名を設定するという使い方もできます。
  • ホスト情報
    • 接続先サーバの情報を「サーバ名/フォルダ」の形式で入力してください。
    サーバ名のみの指定や、深い階層のパスでも指定可能です。
    ここで指定されたパスがルートディレクトリとなり、上のパスには移動できません。
    例）192.168.1.120/comics/雑誌
  • アカウント名
    • サーバ接続時の認証に使用するアカウント名を入力してください。
    認証が不要な場合は空欄にしてください。
  • パスワード
    • サーバ接続時の認証に使用するパスワードを入力してください。
    認証が不要な場合は空欄にしてください。
    現在は接続をURIの指定で行っているため、パスワードに記号を入力するとエラーとなってしまいます。

    サーバへの接続

    リストから接続したいサーバ情報を選択すると接続されます。
