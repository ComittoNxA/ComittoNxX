テキスト表示画面-設定

テキスト表示画面の設定

テキスト表示画面の表示と操作に関する設定です。

起動時状態の設定

サイズ調整方法

ファイルを開いたときのテキストの表示サイズの初期値を選択します。
画面上部の操作エリアでの拡大モードの変更は一時的なものですので、ファイルオープン時に適用したい拡大モードはこちらで設定してください。
  • 元のサイズで表示
    • 用紙サイズのまま表示します。
  • 幅に合わせて表示
    • 用紙の縦横比のまま画面の幅に拡大／縮小して表示します。
  • 高さに合わせて表示
    • 用紙の縦横比のまま画面の高さに拡大／縮小して表示します。
  • 全体を表示
    • 用紙の縦横比のまま画面からはみ出さないように拡大／縮小して表示します。

見開きモード

見開きページをどのように表示するかを選択します。
  • 見開き表示
    • ２ページを並べて表示します。
  • 単ページ表示
    • １ページずつ表示します。
  • 連続表示
    • 全ページを並べて表示します。
    スクロールで続けて読んでいくことができます。

用紙選択

テキストを表示する用紙サイズを選択します。
  • 画面サイズ（縦）
    • 画面サイズを基準にしてテキストを表示します。
  • 800x1280
  • 720x1280
  • 540x960
  • 480x800
    • 指定のサイズを基準にしてテキストを表示します。

表示関連の設定

フォント選択

テキストの描画に使用するフォントをメモリに配置したフォントファイルから選択します。
ComittoN起動時にcomittoフォルダが/sdcardや/mnt/sdcardに作成されますので、その中にfontフォルダを作成し中にフォントファイルをコピーしてください。
フォントファイルは拡張子がttf、otfのもので動作確認しました。

(フォルダの例)
/sdcard/comitto/font/ipaexg.ttf

みかちゃんフォント、IPAexフォントなどで動作を確認しました。
  • みかちゃんフォント
    • http://www001.upp.so-net.ne.jp/mikachan/
    Download→Windows用→OpenTypeFont
  • IPAex
    • http://ossipedia.ipa.go.jp/ipafont/index.html
    ダウンロードページ→TTFファイル/TTF File

挿絵のサイズ

挿絵を表示する際のサイズを設定します。

通知領域を非表示にする

Androidの通知領域を非表示にします。
イメージ表示画面を一旦閉じたあと有効になります。

表示方向

イメージ表示画面を表示する向きを設定します。
カメラボタンによる回転は縦固定又は横固定の場合のみ動作します。
  • 回転あり
    • 本体の向きに合わせて自動的に画面を回転します。
  • 縦固定
    • 本体の向きに関係なく画面を縦向き表示に固定します。
  • 横固定
    • 本体の向きに関係なく画面を横向き表示に固定します。
  • 縦固定（90°回転）
    • Android3.x以降のステータスバーが消せない端末用の設定です。
    画面を縦向き表示に固定しますが、アプリによる描画や操作は横向きの状態となります。
    この状態で端末を横持ちするとステータスバーが右側に表示になるため、画面の高さ全てを画像の描画に利用できます。

中央に余白を表示

「並べて表示」で２枚の画像を表示しているとき間に余白を空けます。
余白の幅は詳細設定の「中央の余白幅」で設定可能です。

中央に影を描画

「並べて表示」で２枚の画像を表示しているとき中央にページの中心っぽい影を描画します。
影の幅は詳細設定の「中央の影の幅」で設定可能です。

ページめくりのエフェクト

表示するページを変更したとき新しいページが横からスクロールして描画されます。

音量キーでのスクロール量（幅）

音量キーの１回の操作で横方向にスクロールする最大の長さを設定します。

音量キーでのスクロール量（高さ）

音量キーの１回の操作で縦方向にスクロールする最大の長さを設定します。

サイズ設定

見出しのフォントサイズ

本のタイトル、著者のフォントサイズ。

テキストのフォントサイズ

本文のフォントサイズ。

ルビのフォントサイズ

ルビ（ふりがな）のフォントサイズ。

ヘッダ/フッタのフォントサイズ

ページ上部のタイトルと下部のページ番号のフォントサイズ。

行間のサイズ

本文の行間（横方向）の間隔をドットで指定。

文字間のサイズ

本文の文字間（縦方向）の間隔をドットで指定。

左右の余白サイズ

用紙左右の余白サイズをドットで指定。

上下の余白サイズ

用紙上下の余白サイズ（タイトルとページ数の高さは除いた部分）をドットで指定。
