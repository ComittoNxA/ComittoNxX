Common Settings

Common setting file selection screen and the image display screen.

Charset

Sets the character code which should be treated as the file names in the RAR or ZIP file.
  • UTF-8
  • Shift_JIS
  • EUC-JP

Not show hidden files in Mac OS X

Is set to hide hidden files in OS files that begin to treat as part of a ".".

Button to rotate the screen

Set the button to rotate the screen orientation in the display screen and image file selection screen.
Difficult because there are several press the shutter button and the presence or absence of focus by the terminal button, please set the buttons easy to operate.
