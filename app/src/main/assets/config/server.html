How to operate the Server Select

This section describes how to operate the server selection screen.

Registration of the server

Setting screen will appear when you press and hold the item you want to set position in the list, please enter the information for the server.
Content is set if the list of configured items will be displayed.

  • Display Name
    • This is the name to be used only for display to the list name.
    You can also use if you have not set this item to the list will be displayed as "undefined", to set a display name server only if you use if you do not want to show to people.
  • Host
    • Please enter in the format "server/folder" information of the server to connect to information.
    You can specify only the server name specified or, in the path of deep hierarchy.
    The path specified here is the root directory, can not move to parent.
    example) 192.168.1.120/comics/magazine
  • Account Name
    • Please enter the account name used to authenticate when connecting to the server.
    If you do not need authentication, please leave blank.
  • Password
    • Please enter the password used to authenticate when connecting to the server.
    If you do not need authentication, please leave blank.
    Because now we can specify the connection URI, it will become a symbol error if you enter the password.

    Connection to the server

    Is connected with the server to select the information you want to connect from the list.
