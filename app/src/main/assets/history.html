Features

  • Supported e-book file formats:
    • PDF, EPUB, TEXT, Aozora Bunko
  • Supported image file formats:
    • JPEG, GIF, PNG, WebP, AVIF, HEIF (Android 8.0 or higher), JXL
  • Supported compressed file formats:
    • ZIP, RAR
  • Supported shared online storage services
    • Windows file sharing (SMB)
    • Access to shared online storage via external apps
      • Storage Access Framework (SAF), File Picker
  • Supports Windows file sharing (SMB)

Notes

  • If you are using a 32-bit application or a 32-bit OS, please reduce the memory usage.
  • If you use a domain in the SMB connection information, the initial connection may be slow.
  • Analyzing the structure of RAR compressed files on online storage takes time. If you want to analyze file names when creating thumbnails, we recommend using ZIP compressed files.

Known issues

  • The contents of the in-app help are out of date.
  • Unable to delete or rename local files on Android 10.
