Features

  • Supported e-book file formats:
    • PDF, EPUB, TEXT, Aozora Bunko
  • Supported image file formats:
    • JPEG, GIF, PNG, WebP, AVIF, HEIF (Android 8.0 or higher), JXL
  • Supported compressed file formats:
    • ZIP, RAR
  • Supports Windows file sharing (SMB)
